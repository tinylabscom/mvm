## Quickstart (3 Steps)

### 1. Install All 5 Skills

```bash
cp -r Skills/*/ ~/.openclaw/skills/
openclaw skills list
openclaw skills check
```

Verify they loaded:

```bash
openclaw skills list
```

You should see: `content-writer`, `web-researcher`, `calendar-manager`, `email-drafter`, `file-organizer`.

### 2. Start Using

Send your agent a message:

```
Write a LinkedIn post about why local AI agents beat cloud automation
```

```
Research the top 5 email marketing tools for solopreneurs
```

```
Organize my Downloads folder
```

That's it. Your agent handles the rest.

### 3. Connect a Channel (Optional)

Want to talk to your agent from your phone? Set up Telegram or Discord.
See `Going Deeper/Channels/` in your Setup Kit for guides.

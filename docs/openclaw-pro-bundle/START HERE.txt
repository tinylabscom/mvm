OPENCLAW SETUP KIT - START HERE
===============================

This kit is built for complete beginners.
Goal: get OpenClaw running fast on your own machine, with your own key.

IMPORTANT
- You need an Anthropic API key: https://console.anthropic.com/settings/keys
- Your key stays on your machine/server. We never ask you to send private keys.

--------------------------------------------------
OPTION A - MAC (easiest)
--------------------------------------------------
1) Open this folder
2) Double-click Setup.command
3) If macOS blocks it: right-click Setup.command -> Open
4) Follow the prompts
5) Dashboard opens in your browser when done

--------------------------------------------------
OPTION B - LINUX VPS (ADVANCED, NOT BEGINNER)
--------------------------------------------------
If this is your first OpenClaw setup, do the Mac setup first.

1) Upload this folder to your VPS
2) SSH into your VPS and cd into this folder
3) Run:
   bash Setup-VPS.sh
4) Follow the prompts
5) Script prints an SSH tunnel command at the end
6) Run that tunnel command on your local computer
7) Open http://127.0.0.1:18789 in your local browser

--------------------------------------------------
WHAT THE SCRIPT DOES
--------------------------------------------------
- Installs OpenClaw using the official installer
- Installs Node.js if required
- Runs onboarding with safe defaults
- Creates SOUL.md, USER.md, MEMORY.md
- Starts/restarts the gateway

--------------------------------------------------
IF YOU GET STUCK
--------------------------------------------------
1) Run the setup script again
2) Run diagnostics:
   bash "Bump/OpenClaw Rescue Kit/02-diagnostic.sh"
3) Official docs:
   https://docs.openclaw.ai
